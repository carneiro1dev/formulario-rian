# Projeto Portfólio

## Visão Geral

Este projeto é um portfólio pessoal desenvolvido para apresentar minhas informações profissionais, formação e projetos como desenvolvedor em início de carreira na área de tecnologia. O objetivo principal é centralizar meus contatos, trabalhos e links em um único site simples, funcional e acessível.

## Objetivo do Projeto

Criei este portfólio como uma forma prática de aplicar meus estudos em desenvolvimento web, principalmente na estruturação de páginas, organização de arquivos e interação básica com o usuário. Além disso, o projeto funciona como um ponto central para acompanhar minha evolução técnica.

## Tecnologias Utilizadas

- HTML
- CSS
- JavaScript
- Formspree

## Decisões Técnicas

Como ainda estou em fase inicial nos estudos de backend, optei por utilizar um serviço externo para viabilizar o envio real de mensagens pelo formulário de contato. Essa decisão permitiu manter o projeto funcional enquanto sigo evoluindo nos estudos de integração com servidores e banco de dados.

## Aprendizados

- Estruturação de sites em múltiplas páginas
- Manipulação do DOM com JavaScript
- Estilização responsiva básica
- Integração com serviços externos para envio de dados

## Próximos Passos

- Implementar backend próprio para o formulário de contato
- Integrar banco de dados para armazenamento de mensagens
- Criar painel administrativo para gerenciamento de contatos
- Melhorar acessibilidade e desempenho do site
